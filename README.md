# TradeForGod Sys
https://youtu.be/s8VAZ5kWBNI

This is a system made with PHP.

TradeForGod is a web system developed using PHP with an MVC architecture. It is designed 
to manage users, groups, and internal processes with role-based access control.

This project is currently being improved and migrated to Laravel to make it more scalable 
and easier to maintain.

## Features
- Role-based system (Administrator, Teacher, Student)
- Discord integration for user management
- Authentication and access control
- Contact system for user communication

## Tecnologies
- PHP
- MySQL
- Laravel (in progress)
